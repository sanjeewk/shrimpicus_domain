# Features Roadmap

This document tracks planned features for Shrimpicus.

> **Legend:** (lp) = low priority / later phase

---

## 1. Database & Data Model

### 1.1 To-Dos
- Auto-categorize into types:
  - Adulting tasks (laundry, cooking, etc.)
  - Passion projects
  - Job applications
  - Fitness

### 1.2 Recurring Reminders
- Monthly intervals
- Weekly intervals
- Yearly intervals

### 1.3 Task Metrics
- Track number of completed tasks

### 1.4 Habit Tracker
- Submit daily/weekly progress ("what you did today/this week")
- Visualize lock-in progress

### 1.5 Major Milestones (lp)
- Track and celebrate major achievements

---

## 2. Discord Bot

### 2.1 Voice to Speech
- Accept voice note attachments
- Transcribe and process

### 2.2 Chat Interface
- Natural language commands to add todos
- Natural language reminders
- Natural language habit logging

### 2.3 Polls
- Send yes/no questions
- Send multi-option polls

---

## 3. Web Interface

### 3.1 Notion-Style Kanban Board
- Columns: **To Do**, **Doing**, **Blocked**
- Drag-and-drop task movement

### 3.2 Flower Field Visualization (lp)
- One flower per completed task

### 3.3 Habit Tracker Page
- Visual habit tracking interface
- _(details TBD)_

---

## 4. LLM Features

### 4.1 Database Access
- Direct task add/remove via MCP or RAG
- Full read/write access to all data models

### 4.2 Encouragement System
- Send morning message with today's todos

### 4.3 Self-Regulation Support
- Detect when todo list exceeds 10 items
- Send self-regulation tips and strategies

### 4.4 Journal Prompts
- Send monthly reflection prompts

### 4.5 Weekly Summary
- Generate and send weekly progress recap

---

## 5. Social Features

### 5.1 Multi-User Accounts
- Each user has their own account

### 5.2 Friend Activity Notifications
- Notify when friends complete tasks
- Notify when friends complete multiple tasks in one day

### 5.3 Habit Completion Sharing
- Get notified when friends complete habits (gym, sports, etc.)

### 5.4 Weekly Social Summaries
- Share progress with friend group

### 5.5 Unified Field Visualization (lp)
- Combined flower field showing all friends' completed tasks
